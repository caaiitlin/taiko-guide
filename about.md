# About

taiko-guide is a community-driven documentation project for Taiko no Tatsujin.

The goal of this project is to provide clear, comprehensive, and accessible English-language documentation covering all aspects of the game, from gameplay fundamentals and arcade features to home controllers, DIY projects, and advanced techniques.

Information is often scattered across forum posts, Discord servers, social media, YouTube, and Japanese websites. This project aims to bring that knowledge together into a single, easy-to-navigate resource.

This website is an ongoing project and will continue to grow as new guides and information are added.
